# 🐉 FlapFury

FlapFury is a simple yet addictive 2D flying game built using **Unity** and **C#**, inspired by the classic Flappy Bird mechanics.

## 🎮 Gameplay

Control Toothless the dragon and fly between obstacles (pillars) to survive as long as possible. The game ends if you hit a pillar — but you can always restart and try again!

## 🛠️ Features

- 🐉 2D side-scrolling flying mechanics
- 🚀 Rigidbody2D physics and collision detection
- 🧠 Score tracking and display
- 💥 Game Over screen with restart functionality
- 🎨 Custom character sprite (Toothless)
- ⌨️ Controls: Press `Space` to flap

## 📦 Built With

- [Unity 2D](https://unity.com/)
- C# (MonoBehaviour scripting)
- Unity’s new Input System (with legacy fallback)
